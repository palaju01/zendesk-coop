# Installation

- Please install node v14.16.0 (https://nodejs.org/en/)

# Front-end (React.js)
From the root folder (ZENDESK-COOP) run:
```
npm install
npm start
```

# Back-end (Node.js)
From the root folder (ZENDESK-COOP) run:
```
cd server
npm install
node server.js
```

# Open the app

Visit http://localhost:3000/zendesk-tickets
